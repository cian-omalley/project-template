# Decision Log

Use this file to record important decisions and why they were made.

## Decision Template

```md
## YYYY-MM-DD — Decision Title

**Status:** Proposed / Accepted / Rejected / Superseded

**Context:**
What situation led to this decision?

**Decision:**
What did we decide?

**Reasoning:**
Why this option?

**Consequences:**
What changes because of this?
```

---

## YYYY-MM-DD — Project Initialized

**Status:** Accepted

**Context:**
A reusable project folder was needed.

**Decision:**
Use this structured template as the project base.

**Reasoning:**
It separates brief, planning, research, design, development, documentation, delivery, archive, and knowledge-base work.

**Consequences:**
All new project work should be organized into the matching folders.
